
# Villa Ana – Web stranica

## Opis projekta
Web stranica za predstavljanje luksuzne vile "Villa Ana".  
Stranica omogućuje pregled smještaja, informacija o vili te kontakt i slanje upita za rezervaciju.

---

## Stranice
- **index.html** – Naslovnica (uvodni prikaz i izdvojene slike)
- **o-nama.html** – Informacije o vili i njenoj priči
- **smjestaj.html** – Pregled smještaja i pogodnosti
- **kontakt.html** – Kontakt forma i osnovne informacije

---

## Kako pokrenuti projekt
1. Preuzmi ili kloniraj projekt
2. Otvori mapu u VS Code-u
3. Pokrentanje pomoću **Live Server** ekstenzije  

---

## Tehnologije
- HTML
- CSS  
  - Flexbox  
  - Grid  
  - CSS varijable  
- Google Fonts (Playfair Display, Inter)

---

## Funkcionalnosti
- Responsive dizajn (prilagodba za mobitele i tablete)
- Navigacija između stranica
- Galerija slika
- Kontakt forma za upite
- Ugrađena Google mapa
